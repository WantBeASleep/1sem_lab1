# Лаба №1 
____
## Условие лаб.работы
Проверить делимость целого числа n на 7. Число делится на 7, если результат вычитания удвоенной младшей цифры числа из числа десятков в записи числа делится на 7.

````

    378 Delitsya na 7
````
